# Prototype-ESP32-Object-Detection-Navigation-Robot
This project is an ESP32-based Object Detection and Navigation Robot prototype. The robot detects nearby objects, estimates direction, and maps room boundaries. Sensor data is displayed through a web interface for monitoring and analysis.

Hi! I'm Raj Madheshiya, a 15-year-old student from India interested in Robotics, Electronics, IoT, and CAD Design.

This project is an ESP32-based Object Detection and Navigation Robot prototype. The robot detects nearby objects, estimates direction, and maps room boundaries. Sensor data is displayed through a web interface for monitoring and analysis.

Audio feedback using DFPlayer Mini
ESP32 Wi-Fi connectivity
Modular design for future upgrades
